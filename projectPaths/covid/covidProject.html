Covid-19 News and Stat Viewer

  • Implemented API calls and endpoints with Node.js and Express.
  • Earth Visualizer implemented with miniatureEarth.js.
  • Used click event-listener on Earth to pass country IDs to endpoints and return data.
  • Used Mongoose and MongoDB to allow for user news contributions.
  • Rendered information with JavaScript DOM manipulation.
Check it out here at: covid-news-viewer.herokuapp.com
